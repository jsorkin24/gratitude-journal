## Installation

### For Developers
Clone the source locally:

```sh
$ git clone https://github.com/jsorkin24/calendar-web-app.git
```

Use your package manager to install `npm`.

Install project dependencies:

```sh
$ npm install
```
Start the app:
```sh
$ mongod
```

```sh
$ nodemon
```

### Build Installers
Build app
```sh
$ gulp
```
